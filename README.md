# TodoApp-Handson

シンプルなタスク管理アプリケーションです。

## 概要

ブラウザ上で動作するタスク管理アプリケーションです。タスクの追加・完了・削除といった基本的な操作を行え、データはブラウザの LocalStorage に保存されるため、サーバーやデータベースを用意しなくても利用できます。

## 技術スタック

| 種別 | 使用技術 |
| --- | --- |
| マークアップ | HTML |
| スタイリング | CSS |
| ロジック | JavaScript |
| データ保存 | LocalStorage |

### 各技術の役割

- **HTML**: アプリケーションの構造・画面要素を定義します。
- **CSS**: レイアウトやデザインなどの見た目を整えます。
- **JavaScript**: タスクの追加・完了・削除などの動的な処理を実装します。
- **LocalStorage**: タスクデータをブラウザ内に永続化し、リロード後もデータを保持します。

## 特徴

- サーバー不要、ブラウザのみで完結
- データは LocalStorage に保存され、再訪問時も保持される
- 軽量でシンプルな構成
